      olarak ticari kazanç sağlamak amacıyla kullanılamaz. Ticari
      kullanım; müşterilere ücretli hizmet sunmak, yazılımı bir ürün
      veya hizmetin parçası olarak yeniden dağıtmak ya da kurumsal
      ağ yönetimi hizmeti satmak gibi durumları kapsar.

      Ticari kullanım için önceden yazılı izin alınması zorunludur.
      İletişim: https://github.com/burakaslann/LeoNetPro

   kullanılması, Ookla'nın koşullarını da ihlal edebilir.

3. YAZILIM NEDİR / NE DEĞİLDİR

   LeoNet Pro; internet hız testi, ağ tanılaması, Wi-Fi tarama ve
   raporlama işlemleri için tasarlanmış bir masaüstü yardımcı programdır.

   Bu Yazılım:
   - Herhangi bir sunucuya kişisel veri göndermez.
   - Kullanıcı hakkında hiçbir bilgiyi kayıt altına almaz veya üçüncü
     taraflarla paylaşmaz.
   - Arka planda reklam, izleme veya analitik kodu çalıştırmaz.
   - İnternet bağlantısını yalnızca kullanıcının başlattığı testler için
     kullanır.

   Yazılım aşağıdaki ağ işlemlerini gerçekleştirir (yalnızca kullanıcının
   açıkça başlattığı işlemler):
   - Hız testi sunucularına bağlantı (Speedtest CLI — Ookla)
   - Cloudflare CDN üzerinden yedek hız testi (speed.cloudflare.com —
     yalnızca Speedtest CLI bulunmadığında, yalnızca manuel tetikleme,
     oran sınırlı, otomatik/zamanlanmış testlerde kullanılmaz)
   - Cloudflare CDN üzerinden bufferbloat testi sırasında geçici dosya
     indirme/yükleme (tek seferlik, tanılama kapsamında)
   - LibreSpeed tarayıcı tabanlı hız testi (kullanıcının belirttiği URL)
   - iPerf3 istemci/sunucu bağlantıları (kullanıcının tanımladığı adresler)
   - ICMP ping, DNS sorguları, TCP/UDP port kontrolleri ve traceroute
   - DNSSEC doğrulama ve DNS over HTTPS/TLS erişim kontrolü
     (cloudflare-dns.com üzerinden tek seferlik sorgular)
   - DNS leak testi (whoami.akamai.net üzerinden tek DNS sorgusu)
   - GitHub Releases API üzerinden güncelleme kontrolü (isteğe bağlı)

3. WHAT THE SOFTWARE IS / IS NOT

   LeoNet Pro is a desktop utility designed for internet speed testing,
   network diagnostics, Wi-Fi scanning, and reporting.

   This Software:
   - Does NOT send any personal data to any server.
   - Does NOT record or share any information about the user with
     third parties.
   - Does NOT run any advertising, tracking, or analytics code in
     the background.
   - Uses an internet connection only for tests explicitly initiated
     by the user.

   The Software performs the following network operations (only when
   explicitly initiated by the user):
   - Connections to speed test servers (Speedtest CLI — Ookla)
   - Cloudflare CDN fallback speed test (speed.cloudflare.com —
     only when Speedtest CLI is not available, manual trigger only,
     rate-limited, not used in scheduled/automated tests)
   - Cloudflare CDN temporary download/upload during bufferbloat
     diagnostic (single-use, within diagnostics scope)
   - LibreSpeed browser-based speed test (user-specified URL)
   - iPerf3 client/server connections (to user-defined servers)
   - ICMP ping, DNS queries, TCP/UDP port checks, and traceroute
   - DNSSEC validation and DNS over HTTPS/TLS accessibility checks
     (via cloudflare-dns.com, single-use queries)
   - DNS leak testing (single DNS query via whoami.akamai.net)
   - Update check via GitHub Releases API (optional)
